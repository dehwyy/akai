# Akai


## Functions' graphic visualizer

### Requirements:

- CMake
- Any C++ compiler

### Building
```sh
mkdir build                                                                           # build dir
cd build && cmake ..                                                                  # init CMake
cmake --build .                                                                       # build proj
./build/<output.file>                                                                 # run app
```
