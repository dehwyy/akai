## Computer Science Project 2 semester.

## Building
```sh
mkdir build
cd build && cmake ..  # init CMake

cmake --build build   # build
./build/<output.file> # run
```
Or with `make`:y
```sh
make init # Init CMake
make r    # Rebuild and Run
```
